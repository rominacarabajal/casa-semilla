CASA SEMILLA · Landing Premium (lista para Netlify)

1) Subir a Netlify
- Descomprimí este ZIP.
- Arrastrá TODA la carpeta a Netlify (Deploys → Drag & drop).
- No subas una carpeta vacía: deben quedar index.html y styles.css en la RAÍZ del deploy.

2) Reemplazar imágenes
Carpeta /img
- hero-fachada.jpg (portada)
- galeria-patio.jpg
- galeria-interior.jpg
- etapa1-planta.jpg
- etapa2-planta.jpg (placeholder)
- etapa3-planta.jpg (placeholder)

3) Configuración rápida
Editá site.config.json
- WhatsApp y link MercadoPago
- Precios
- access_key y download_file

4) Entrega del proyecto
Reemplazá 'casa-semilla-proyecto.zip' por tu ZIP real (mismo nombre).
TIP: para máxima protección, cifrá el ZIP con contraseña con 7-Zip/WinRAR.
